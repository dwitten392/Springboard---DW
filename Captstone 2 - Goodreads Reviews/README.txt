This project uses a subset of goodreads reviews and variety of NLP techniques to perform sentiment analysis on the text. In this folder you will find the code (ipython format), a report (PDF), and slides(pptx) on this project.

Please reference Mengting Wan, Rishabh Misra, Ndapa Nakashole, Julian McAuley, "Fine-Grained Spoiler Detection from Large-Scale Review Corpora", in ACL'19.

https://github.com/MengtingWan/goodreads
